The Runaway Train: In active Development

The Runaway Train is a walking simulator set in a dystopian world. The game is currently in active development, and it is a passion project of mine. The game features an immersive storyline that explores themes of Time travel, hope and Trauma. Being developed in unity, it has it's own Time travel system that is expandable and modular. currently working on a system to create predictable time loops and paradoxes with the unity state machine.
